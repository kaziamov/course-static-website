# Введение в JavaScript

JavaScript - это мощный язык программирования, который используется для создания интерактивных веб-страниц и веб-приложений.

## Что такое JavaScript?

JavaScript был создан в 1995 году Бренданом Эйхом всего за 10 дней! Сегодня это один из самых популярных языков программирования в мире.

### Основные особенности:

- **Интерпретируемый язык** - код выполняется напрямую браузером
- **Динамическая типизация** - типы переменных определяются автоматически
- **Функциональный и объектно-ориентированный** - поддерживает разные парадигмы программирования

## Первая программа

Давайте напишем классическую программу "Hello, World!":

```javascript
console.log("Hello, World!");
```

Этот код выводит текст в консоль браузера.
